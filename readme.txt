IDE used:
IntelliJ IDEA Community Edition 2018.1.5

Java version:
1.8.0_05

Working directory relative to repo:
csc413-tankgame-reneelias\TankGamesGDXProject\core\assets

Run the game:
1) Open an instance of Command Prompt
2) Navigate to the repo directory
3) Navigate to the jar folder in the repo
4) Run the following command to run the jar file:
java -jar TankGamesGDXProject.jar

Controls for the Game:
Player 1:
Forward = w
Backward = s
Rotate left = a
Rotate right = d
Shoot = space

Player 2:
Forward = up arrow
Backward = down arrow
Rotate left = left arrow
Rotate right = right arrow
Shoot = right ctrl
